# PI Web

Frontend for the PI project, built with React, TypeScript, and Vite.

## Local Development

```bash
npm install
cp .env.example .env
npm run dev
```

For local backend development, set:

```text
VITE_API_BASE_URL=http://127.0.0.1:8000/
```

## Build
